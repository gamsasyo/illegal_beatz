# STREET SESSION — 설계 문서

토니 호크 프로 스케이터(90s~2000s 초) 감성의 초기 3D 스케이트 묘기 게임. 단일 HTML + Three.js, 브라우저 실행.

## 1. 컨셉 / 룩
- 초기 3D(PS1) 감성: 로우폴리 + 플랫 셰이딩, 안개(fog), 제한 팔레트
- 내부 저해상도 렌더 → 확대(픽셀 뭉개짐), 선택적 버텍스 흔들림(PS1 wobble)
- flight.html 패턴 계승: Three.js 0.160 importmap, module script, EffectComposer/bloom 가능

## 2. 조작 (WASD + SPACE)
```
지상:  W 가속 / S 브레이크·후진 / A·D 회전(스티어링) / SPACE 올리(점프)
공중:  A·D = 회전·플립계(SPIN / KICKFLIP),  W·S = 그랩계(INDY / NOSEGRAB)
       → 누를 때마다 +점수, 콤보 배수 ++
레일:  레일/엣지 위 착지 시 자동 그라인드, 점수 누적, SPACE로 튀어나가 콤보 유지
```

## 3. 트릭 / 점수 (① 올리 ② 공중트릭 ③ 그라인드 ⑤ 콤보배수)
- 공중·그라인드로 트릭 이어가면 콤보 누적 → HUD에 콤보 문자열 + 배수
- 깔끔 착지 = 콤보점수 × 배수 → 총점 입금(BANK!)
- 삐끗(트릭 회전 중 바닥에 처박힘) = 콤보 소멸 (위험/보상)
- 매뉴얼(④)은 v1 제외
- 세션: 프리스케이트, 시간 제한 없음, 하이스코어 갱신만

## 4. 레벨 (런던 사우스뱅크 — OSM 하이브리드)
```
배경 = 실제 런던 사우스뱅크 거리·건물 (OpenStreetMap, 가입/키 불필요)
       Overpass API로 footprint+높이 받아 → extrude → 로우폴리 박스
그 위 = 손으로 까는 스케이트 스팟 (뱅크·렛지·레일·기둥·램프)
바닥 = 평평한 그라운드 평면 + 거리 텍스처/색
```
- 한 구역(블록) 단위, 루프 가능하게
- OSM 데이터는 fetch 실패/오프라인 대비해 fallback(절차적 더미 블록) 둠

## 5. 컴포넌트 경계
- `World` : OSM 로드/extrude + 스케이트 스팟 배치 + 콜라이더 등록
- `Skater` : 아케이드 물리(가속/스티어/중력/올리), 상태(ground/air/grind)
- `TrickSystem` : 입력→트릭 판정, 콤보 누적/배수, 뱅크/바일
- `HUD` : 점수·콤보·속도 표시, 트릭 팝업
- `Renderer/Look` : 저해상도+픽셀화+안개+플랫셰이딩 파이프라인

## 6. 검증 기준
- WASD로 굴러다니고 SPACE로 점프됨
- 공중에서 키 → 트릭명+점수 팝업, 배수 증가
- 레일 위 착지 → 그라인드, SPACE로 탈출
- 깔끔 착지 = BANK, 처박힘 = 콤보 소멸
- 사우스뱅크 OSM 건물이 로우폴리로 뜸(실패 시 fallback)
- HUD에 총점/하이스코어/콤보/속도 표시
